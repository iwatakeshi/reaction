# reaction

A minimalistic React + JSPM boilerplate.

## Workflow

* JSPM
  * Babel (with JSX support) 
* Gulp
  * JSCS - Uses Airbnb Style preset.
  * BrowserSync


## Usage

**Install the global version of JSPM and Gulp**

```bash
sudo npm i jspm gulp -g
```

**Install packages**

```bash
sudo npm i && jspm i
```

** Build and Watch **

```bash
gulp develop
```
